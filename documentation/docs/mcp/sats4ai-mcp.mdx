## Configuration

Sats4AI is a remote streamable-HTTP MCP server — no install or local process required. No API keys or environment variables are needed upfront; each tool call is paid with a Bitcoin Lightning micropayment at request time.

You will need a Lightning wallet with sats loaded (any wallet works: [Phoenix](https://phoenix.acinq.co/), [Wallet of Satoshi](https://www.walletofsatoshi.com/), [Alby](https://getalby.com/), etc.).

## Payment Flow

Every Sats4AI tool follows the same payment pattern:

1. Call `create_payment` with the tool name → receive a Lightning invoice and a `paymentId`
2. Pay the invoice with any Lightning wallet (confirms in seconds)
3. Call `check_payment_status({ paymentId })` → wait for `{ paid: true }`
4. Call the tool with `{ paymentId, ...params }` → receive the result

If a tool call fails after payment, call `request_refund({ paymentId })` to mark it as refundable.

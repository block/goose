# 安装 goose

## 桌面端

请从发布页下载对应平台安装包并完成安装。

## CLI

```bash
curl -fsSL https://github.com/block/goose/releases/download/stable/download_cli.sh | bash
```

Windows 用户可在 PowerShell / Git Bash 中执行，并确认 PATH 配置。

## 验证安装

```bash
goose --version
```

## 下一步

安装完成后请继续[配置 Provider](/docs/getting-started/providers)。

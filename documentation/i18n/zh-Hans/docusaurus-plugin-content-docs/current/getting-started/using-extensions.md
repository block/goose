# 使用扩展

扩展基于 MCP，可为 goose 提供额外工具与上下文。

## 启用方式

- 桌面端：进入 `Settings` 或 `Extensions` 页面启用。
- CLI：执行 `goose configure`，选择 `Add Extension`。

## 建议

- 只启用当前任务需要的扩展。
- 对高风险工具使用权限确认。

更多内容见[工具权限](/docs/guides/managing-tools/tool-permissions)。

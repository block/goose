# 5 分钟上手 goose

goose 是一个可扩展的开源 AI agent，可自动化软件开发任务。

## 你将完成

- 安装 goose（桌面端或 CLI）
- 配置 LLM Provider
- 启动第一个会话
- 启用一个 MCP 扩展

## 安装

请参考[安装指南](/docs/getting-started/installation)。

## 配置 Provider

首次启动后，选择一个 Provider（例如 Tetrate Agent Router），并完成认证或 API Key 配置。

详细说明见[Provider 指南](/docs/getting-started/providers)。

## 启动会话

- 桌面端：完成向导后进入 Home 即可开始。
- CLI：执行 `goose session`。

## 启用扩展

前往 `Extensions`，启用 `Computer Controller`，然后回到会话中继续。

# 环境变量

环境变量可用于快速切换 Provider、模型、路径与运行行为。

## 常用变量

- `GOOSE_PROVIDER`
- `GOOSE_MODEL`
- `GOOSE_PATH_ROOT`
- Provider 凭证变量（如 `ANTHROPIC_API_KEY`）

## 建议

- 在 CI 中使用密钥管理系统注入。
- 本地开发避免把密钥提交到仓库。

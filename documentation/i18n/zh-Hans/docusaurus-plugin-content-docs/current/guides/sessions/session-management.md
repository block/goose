# 会话管理

## 你可以做什么

- 创建新会话
- 恢复已有会话
- 删除不再需要的会话
- 导出会话记录

## CLI 示例

```bash
goose session
goose session -r
goose session list
goose session remove
```

## 建议

- 按项目/任务拆分会话。
- 长任务定期总结，减少上下文噪音。

# goose CLI 命令

## 常用命令

```bash
goose configure
goose session
goose run --text "..."
goose session list
goose web --open
```

## 建议实践

- 先 `configure`，再 `session`。
- 自动化场景优先使用 `run`。
- 通过 `session list` 管理历史会话。

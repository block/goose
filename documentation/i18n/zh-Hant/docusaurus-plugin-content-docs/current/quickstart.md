# 5 分鐘上手 goose

goose 是可擴展的開源 AI agent，可自動化軟體開發任務。

## 你將完成

- 安裝 goose（桌面版或 CLI）
- 設定 LLM Provider
- 啟動第一個工作階段
- 啟用一個 MCP 擴展

## 安裝

請參考[安裝指南](/docs/getting-started/installation)。

## 設定 Provider

首次啟動後，選擇一個 Provider（例如 Tetrate Agent Router），並完成驗證或 API Key 設定。

詳情請見[Provider 指南](/docs/getting-started/providers)。

## 啟動工作階段

- 桌面版：完成引導後進入 Home 開始。
- CLI：執行 `goose session`。

## 啟用擴展

前往 `Extensions`，啟用 `Computer Controller`，再回到工作階段繼續。

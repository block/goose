# 工作階段管理

## 你可以做什麼

- 建立新工作階段
- 恢復既有工作階段
- 刪除不再需要的工作階段
- 匯出工作階段記錄

## CLI 範例

```bash
goose session
goose session -r
goose session list
goose session remove
```

## 建議

- 依專案/任務拆分工作階段。

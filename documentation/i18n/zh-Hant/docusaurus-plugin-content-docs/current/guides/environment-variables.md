# 環境變數

環境變數可用於快速切換 Provider、模型、路徑與執行行為。

## 常用變數

- `GOOSE_PROVIDER`
- `GOOSE_MODEL`
- `GOOSE_PATH_ROOT`
- Provider 憑證變數（例如 `ANTHROPIC_API_KEY`）

## 建議

- 在 CI 使用密鑰管理系統注入。
- 本機開發避免把密鑰提交到版本庫。

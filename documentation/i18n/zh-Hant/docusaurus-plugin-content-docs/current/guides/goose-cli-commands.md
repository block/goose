# goose CLI 指令

## 常用指令

```bash
goose configure
goose session
goose run --text "..."
goose session list
goose web --open
```

## 建議實務

- 先 `configure`，再 `session`。
- 自動化情境優先使用 `run`。
- 透過 `session list` 管理歷史工作階段。

# 安裝 goose

## 桌面版

請從發佈頁下載對應平台安裝檔並完成安裝。

## CLI

```bash
curl -fsSL https://github.com/block/goose/releases/download/stable/download_cli.sh | bash
```

Windows 使用者可在 PowerShell / Git Bash 執行，並確認 PATH 已設定。

## 驗證安裝

```bash
goose --version
```

## 下一步

安裝完成後請繼續[設定 Provider](/docs/getting-started/providers)。

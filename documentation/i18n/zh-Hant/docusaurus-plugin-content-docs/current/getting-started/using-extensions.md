# 使用擴展

擴展基於 MCP，可為 goose 提供額外工具與上下文。

## 啟用方式

- 桌面版：在 `Settings` 或 `Extensions` 頁面啟用。
- CLI：執行 `goose configure`，選擇 `Add Extension`。

## 建議

- 只啟用目前任務需要的擴展。
- 對高風險工具使用權限確認。

更多內容見[工具權限](/docs/guides/managing-tools/tool-permissions)。

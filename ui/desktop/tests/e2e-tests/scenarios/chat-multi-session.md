# Multi-Session Chat Flow

1. Send "Hello" in the chat input
2. Verify the response "Howdy! I am goose, ready to help." appears
3. Navigate to Home and verify total sessions count is 1
4. Send "What is 2+2?" and verify the response "2 + 2 = 4"
5. Open session list and verify 2 sessions exist
6. Click the second session to switch back to it
7. Verify the previous response "Howdy! I am goose, ready to help." is visible
8. Send "Tell me a joke" in the active session
9. Verify the response "Because light attracts bugs!" appears
10. Verify the cost tracker shows a non-zero value

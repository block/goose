# Recipe Manager: Full CRUD

1. Navigate to Recipes and verify "No saved recipes" is shown
2. Click "Create Recipe" and fill in:
   - Title: "Joke Recipe"
   - Description: "A recipe that tells a joke"
   - Instructions: "You are a comedian"
   - Prompt: "Tell me a joke"
3. Save the recipe and verify it appears in the list
4. Use the recipe — trust and execute it
5. Verify the response "Because light attracts bugs!" appears
6. Navigate back to Recipes and edit the recipe:
   - Change instructions to "You are a math tutor"
   - Change prompt to "What is 2+2?"
7. Save and use the updated recipe
8. Verify the response "2 + 2 = 4" appears
9. Navigate back to Recipes and delete the recipe
10. Verify "No saved recipes" is shown again

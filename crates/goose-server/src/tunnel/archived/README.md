# Archived Tunnel Implementations

This directory contains archived tunnel implementations that are not currently in use but are preserved for future reference.

## Contents

### tailscale-tunnel.sh
The shell script that was used to start Tailscale tunnels. This is from the old frontend TypeScript implementation that actually worked. Kept for reference in case Tailscale support is re-added in the future.

For now, we're focusing on the Lapstone tunnel which works out of the box.
